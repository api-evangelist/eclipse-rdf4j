Milestone number 1 of the upcoming 6.0.0 release of RDF4J is now available for download. RDF4J 6.0.0 is a major release of the RDF4J framework, focusing on dependency upgrades, LMDB improvements, and HTTP client improvements. Please note that RDF4J 6.0.0-M1 requires Java 25.
