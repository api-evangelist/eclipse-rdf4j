Milestone number 2 of the upcoming 5.3.0 release of RDF4J is now available for download. RDF4J 5.3.0 is a minor release focusing on stability and getting the code ready for the next major release. Notable changes in 5.3.0 M2 SHACL validation can now be configured more precisely for inferred data ( GH-5676 ).
